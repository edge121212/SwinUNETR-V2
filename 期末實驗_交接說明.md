# 期末實驗交接說明（SwinUNETR-V2 + Decoder Attention Gate）

> 分支：`feat/decoder-attention-gate`。先 `git pull` 切到這個分支再跑。
> 資料集放在 `dataset/Task05_Prostate/`（沒有的話先 `./run.sh download Prostate`）。

---

## 一、我們改了什麼（一句話）

在 **SwinUNETR-V2 的 decoder skip connection 上加 Attention Gate**（Attention U-Net 式的加性注意力），用來聚焦最難分割的 **PZ（周邊區）**。改造是**可開關、且逐層可選**的，baseline 與改造模型**共用同一份程式碼**，只差一個旗標。

### 動到的檔案
| 檔案 | 改動 |
|---|---|
| `models/swin_unetr.py`（**新增**，從 MONAI 1.5.2 複製來改）| 新增 `AttentionGate` 與 `AttnUnetrUpBlock` 兩個 class；`SwinUNETR` 新增參數 `attn_gate_levels`，依此決定哪幾個 decoder 層套 attention gate |
| `main.py` / `test.py` | import 改成 `from models.swin_unetr import SwinUNETR`；新增 `--attn_gate_levels` 旗標並傳進模型 |
| `run.sh` / `run.bat` | 新增環境變數 `ATTN_LEVELS` 開關；AG 實驗的 `runs/` 資料夾自動加 `_ag<levels>` 後綴，不會蓋到 baseline |

### 關鍵保證
- `attn_gate_levels` 為**空時，模型與原版 MONAI SwinUNETR 逐 key 完全相同**（已用 CPU 驗證），所以**現有 baseline 的 checkpoint 仍相容**。
- decoder 層編號：**1 = 最高解析（靠近輸出）… 5 = 最深/最粗**。
- 全程 **from scratch、不下載任何 pretrained 權重**（與論文一致）；`use_v2=True` 固定開啟。

## 二、三個實驗 + 指令

| 實驗 | 說明 | `ATTN_LEVELS` | 結果存到 |
|---|---|---|---|
| **E0 baseline** | V2，無 AG（對照組）| 不設 | `runs/prostate_fold{0..4}` |
| **E1 主方法** | V2 + AG（全 5 層）| `1,2,3,4,5` | `runs/prostate_ag12345_fold{0..4}` |
| **E2a 消融** | V2 + AG（只高解析層）| `1,2,3` | `runs/prostate_ag123_fold{0..4}` |

> **E0 目前已在另一台（5070Ti）跑中**，你這台**主要跑 E1、行有餘力再跑 E2a**。

### Linux / WSL / macOS（`run.sh`）
```bash
# E1：完整 5-fold，每 fold 500 epoch
ATTN_LEVELS="1,2,3,4,5" ./run.sh kfold Prostate 500

# E2a（選做）
ATTN_LEVELS="1,2,3" ./run.sh kfold Prostate 500

# 不帶 epoch 數 = 自動跑滿 ~40k iterations（論文設定，單 fold 約 10h）
ATTN_LEVELS="1,2,3,4,5" ./run.sh kfold Prostate
```

### Windows（`run.bat`）
```bat
set ATTN_LEVELS=1,2,3,4,5
run.bat kfold Prostate 500
```
（跑別的實驗前先 `set ATTN_LEVELS=` 清掉，或開新視窗。）

## 三、跑法說明（重要）

- **彈性 / 隨時可停**：kfold 會依序對 fold 0~4「訓練→測試」。已經有 `runs/<logdir>_test.log` 的 fold 會**自動跳過**；只訓練到一半（有 `model_final.pt`）的 fold 會**從 checkpoint 續訓**。所以中途關機 / 改天再跑都沒問題，重下同一條指令即可接續。
- **要不要跑滿你自己決定**：500 epoch ≈ 單 fold 3h（5-fold ≈ 15h）；跑滿 40k ≈ 單 fold 10h（5-fold ≈ 52h）。時間夠就跑 40k 衝「贏論文」，趕的話 500 epoch 就能交出完整對照。
- **測試用的權重**：是 `model.pt`（**驗證集最佳**模型，對齊論文用 val 挑模型的做法），不是最後一個 epoch。
- **每 25 epoch** 存一次 checkpoint（`val_every=25`）。
